API USER

Rota Pai: /api
POST /api/create Criação de um novo usuário no sistema

Headers obrigatórios:

Content-Type: application/json

Payload (JSON Body)
{
  "pais": "SEUPAIS",
  "name": "SEUNOME",
  "email": "SEUEMAIL@gmail.com",
  "password": "SUA SENHA"
}
GET /api/busca/ Busca flexível de usuário por múltiplos critérios
Parâmetros de Query (Query Params)
Parâmetro Tipo Descrição
id String / Integer O ID único do usuário cadastrado no banco.
nome String O nome ou parte do nome do usuário que deseja buscar.
GET /api/id/{SEUID} Busca direta e estrita por ID único
Parâmetros de URL (Path Params)
Parâmetro Tipo Descrição
{SEUID} String / Integer Insira o ID do usuário diretamente na URL (ex: /api/id/
